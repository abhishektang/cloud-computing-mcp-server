# Additional Resources

This folder contains supplementary course materials that don't fit into the weekly structure.

## What Goes Here?

- Study guides
- Cheat sheets
- Reference materials
- Supplementary readings
- External resources
- Course syllabus
- Exam guidelines
- Project documentation
- Any other helpful materials

## Organization Tips

You can organize materials here however you like:
- By topic (e.g., `docker-guide.pdf`, `kubernetes-cheatsheet.pdf`)
- By type (e.g., `study-guides/`, `cheat-sheets/`)
- Flat structure with clear filenames

## Examples

```
additional-resources/
├── cloud-computing-cheatsheet.pdf
├── aws-reference-guide.pdf
├── docker-commands.md
├── study-guide-final-exam.pdf
└── course-syllabus.pdf
```

## Usage

All materials in this folder are searchable through the MCP server:
- Use `search-materials` to find specific topics
- Use `get-course-overview` to see what's available
- Ask questions like "What additional resources do I have?"
